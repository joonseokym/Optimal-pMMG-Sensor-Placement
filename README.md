# Optimal pMMG Sensor Placement for Human Knee Extensor Analysis

This repository contains the source code used in the research paper **"Optimal pMMG Sensor Placement for human knee extensor analysis."**

## Contents
- **Data Acquisition:** Scripts used to collect raw pMMG signals during experiments.
- **Post-processing:** Algorithms for signal filtering, feature extraction, and analysis.
